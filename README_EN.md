# ✨ XHS-Auto (Xiaohongshu Automation Tool) | Supported App Version: 9.9.0+

Welcome to **XHS-Auto**, a mobile automation tool built with **OCR recognition + Android Accessibility Service**.

This project is mainly designed for **automation technology learning and experimentation**.
By combining image recognition with configurable behavior parameters, it enables automated interaction with mobile application interfaces.

⚠️ This project is intended **for educational and research purposes only**. Do not use it for activities that violate platform policies.

---

# 🚀 Features

### 🤖 Randomized Interaction Behavior

* Swipe actions use randomized curve trajectories
* Click actions include random pixel offsets
* Reduces fixed and detectable interaction patterns

### 👀 Multi-Page Automation Workflow

Supports automated workflows across multiple pages, including:

* Entering note detail pages
* Browsing comment sections
* Navigating different UI states

### 🔍 90%+ Operations Based on OCR

Most UI interactions rely on **OCR text recognition**, minimizing dependency on Accessibility selectors.

### ⚙️ Configurable Behavior Parameters

Users can customize:

* Swipe count
* Browsing duration
* Random delay intervals
* Interaction probability

### 🪟 Floating Window Log System

Real-time logs displayed through a floating window for easier monitoring and debugging.

### 🔓 No Root Required

Runs on **standard Android devices** without root access.

---

# 🔑 Required Permissions

Before running the script, please grant the following permissions:

* **Accessibility Service**
* **Floating Window Permission**
* **Screenshot Permission**
* **Storage Read/Write Permission**

### ⚠️ Recommended Setting

To minimize unnecessary permission access, it is recommended to disable the **“App List Access” permission** for Xiaohongshu in system settings.
