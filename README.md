# ✨ XHS-Auto (小红书自动化工具) | 适用App版本: 9.9.0及以上

欢迎使用 **XHS-Auto**，这是一款基于 **OCR识别 + 无障碍服务** 构建的移动端自动化工具。  
项目主要用于 **自动化技术学习与实践**，通过图像识别与可配置行为参数，实现对应用界面的自动化操作流程。

本项目仅用于自动化技术研究与学习，请勿用于违反平台规则的行为。

---

# 🚀 项目特点

- 🤖 **随机化操作轨迹**  
  滑动采用随机曲线轨迹，点击采用随机偏移，减少固定操作模式。

- 👀 **多页面自动化流程支持**  
  支持进入笔记详情页、浏览评论区等界面操作流程。

- 🔍 **90%+ 操作通过 OCR 识别完成**  
  大部分操作通过图像识别完成，仅保留必要的无障碍操作。

- ⚙️ **行为参数可配置**  
  支持自定义滑动次数、停留时间等参数。

- 🪟 **悬浮窗实时日志**  
  运行过程实时输出日志，方便调试与监控。

- 🔓 **无需 Root 权限**  
  普通设备即可运行。

---

# 🔑 所需权限

使用前需要授予以下权限：

- **无障碍服务权限**
- **悬浮窗权限**
- **截图权限**
- **设备存储读写权限**

### ⚠️ 建议设置

为减少不必要的应用权限访问，建议在系统设置中关闭 **小红书的「获取应用列表」权限** 。

# 📦 功能列表

## 🔎 功能一：搜索浏览

- 输入关键词自动搜索
- 浏览搜索结果中的笔记
- 进入笔记详情页进行浏览
- 支持浏览评论区
- 随机滑动浏览笔记
- 根据行为参数进行随机操作

适用于：

- **账号兴趣养成**
- **关键词内容浏览**
- **模拟真实用户搜索行为**

---

## 🏠 功能二：首页浏览

- 自动浏览首页推荐流
- 命中兴趣内容时进入详情页
- 随机浏览评论区
- 随机停留时间
- 随机滑动轨迹

适用于：

- **账号兴趣标签培养**
- **模拟真实用户刷首页行为**
- **长期稳定养号**
